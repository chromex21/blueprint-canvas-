## ğŸ’¡ Tips & Tricks

### Mind Mapping Workflow
1. Create center node with main topic
2. Add child nodes around it
3. Connect with arrows
4. Color code by category (future feature)

### Story Planning
1. Use sticky notes for scenes
2. Arrange in timeline
3. Connect story beats
4. Use shapes for act containers

### Tree Structures
1. Start with root node at top
2. Add children below
3. Connect parent to children
4. Use basic nodes for hierarchy

### Free-Form Canvas
1. Mix all node types
2. Create visual diagrams
3. No structure required
4. Pure creative freedom

## ğŸ› Troubleshooting

**Node won't select**
- Make sure Select tool is active
- Click center of node, not edge
- Try clicking again

**Can't create connection**
- Connector tool must be active
- Click node center, not edge
- Click second node to complete

**Dialog won't open**
- Use double-tap, not single click
- Select tool must be active
- Wait brief moment between clicks

**Dragging is jumpy**
- Disable "Snap to Grid" for smooth movement
- Or adjust grid spacing for larger snaps
